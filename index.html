Curso GIT

git init - Tornar uma pasta controlada pelo git

git tag - Ver as versões

git blame - Ver as alterações que cada pessoa fez

git diff v0.0.1 v0.0.1 - Comparar as modificações

git add nomeDoArquivo ou git add . (para adicionar todos) ou git add nomeDoDiretorio - Adicionar os arquivos para serem comitados

git status - Ver o estado dos arquivos

git config --global user.name "João Carlos Fonseca"

git config --global user.email "jcfonsecagit@gmail.com"

git commit -m "Mensagem a ser passada" - Comitar os arquivos

git log - Serve para ver os comites que foram feitos

git whatchanged - Serve para ver os arquivos que foram alterados e git whatchanged -p mostra quais partes foram adicionadas e escluidas

git remote - Mostra quais branches remotos o repositorio local tem

git remote add origin https://github.com/pedrosouza18/curso-git.git - Adiciona um repositório remoto

git push origin(repositorio remoto) master(branch a ser criado ou ja criado)

git clone https://github.com/pedrosouza18/curso-git.git - Baixar repositório
